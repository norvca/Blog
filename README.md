# Linux 命令行使用教程

## 1.常用指令

|  指令   |        含义        |
| :---: | :--------------: |
|  cd   |       切换目录       |
|  ls   |     查看文件与目录      |
|  pwd  |      显示当前目录      |
| mkdir |       创建目录       |
| touch |      创建空文件       |
| echo  |     创建带有内容文件     |
|  cp   |        拷贝        |
|  mv   |      重命名或移动      |
|  rm   |        删除        |
| rm -r | 递归删除（可以删除子目录和文件） |
| rm -f |       强制删除       |

## 2.一些技巧
|     指令     |      含义       |
| :--------: | :-----------: |
|    cd -    |    回到上一个目录    |
|     ↑      |    返回上一次指令    |
|     !!     |    使用上一次指令    |
|  alt + .   |   使用最近一个参数    |
| xxx ; yyy  |   一次执行两个命令    |
| xxx && yyy | 一次执行两个命令(与逻辑) |
